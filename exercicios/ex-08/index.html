Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em Negrito usando a tag <B>(Não Semântica).

Nesta frase temos um termo em Destaque usando a tag <strong>(Semântica).

Italico / Ênfase

Nesta frase temos um termo em Itálico usando a tag <I>(Não Semântica).

Nesta frase temos um termo em Ênfase usando a tag <EM>(Semântica).

Texto Marcado

Podemos tambem criar um texto marcado usando a tag <Mark.>

Texto Grande e Pequeno

Texto Deletado

Podemos marcar um texto como excluido, para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi inserido depois.
